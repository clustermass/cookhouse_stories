API Endpoints

users

  • GET /api/users - returns the user information
  • POST /api/users - sign up

session

  • POST /api/session - log in
  • DELETE /api/session - log out

recipes

  • GET /api/ - returns all recipes
  • GET /api/recipes/:id - returns recipe
  • POST /api/recipe - creates a recipe
  • PATCH /api/recipes/:id - edit a recipe
  • DELETE /api/recipes/:id - remove a recipe

comments

  • POST /api/recipes/:recipesId/add - add comment to recipe

Frontend Routes

  • /recipes/new - create a recipe
